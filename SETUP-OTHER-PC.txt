AI Resume Analyzer - Setup Guide (Other PC)
==========================================

Goal:
- Aap pura folder zip karke dusre system me bhej rahe ho.
- Ye guide follow karke project easily run ho jayega.

Required Software
-----------------
1) Python 3.9.x (Recommended)
2) MySQL Server (running)
3) VS Code / Any Editor


Step 1: Extract ZIP
-------------------
1) Zip file extract karo (e.g. Desktop par)
2) Folder name: AI-Resume-Analyzer-main\AI-Resume-Analyzer-main


Step 2: MySQL Setup
-------------------
1) MySQL start karo
2) Database create (agar nahi hai to):
   - CREATE DATABASE cv;

3) DB credentials App me env vars se override ho sakte hain:
   - DB_HOST (default localhost)
   - DB_USER (default root)
   - DB_PASSWORD (default empty)
   - DB_NAME (default cv)

Example (Windows PowerShell):
- setx DB_USER "root"
- setx DB_PASSWORD "your_password"
- setx DB_NAME "cv"

Note: setx ke baad naya terminal open karo.


Step 3: Create Virtual Environment
----------------------------------
Project root me terminal open karo:
Path: AI-Resume-Analyzer-main\AI-Resume-Analyzer-main

Command:
python -m venv venvapp

Activate:
venvapp\Scripts\activate


Step 4: Install Python Packages
-------------------------------
App folder ke requirements install karo:
cd App
pip install -r requirements.txt

Extra packages (updated features ke liye):
pip install scikit-learn joblib scipy reportlab


Step 5: Download NLP Resources
------------------------------
Run these:
python -c "import nltk; nltk.download('stopwords'); nltk.download('punkt')"

SpaCy model (pyresparser ke liye important):
python -m spacy download en_core_web_sm


Step 6: Run the Streamlit App
-----------------------------
Ensure working directory `App` ho:
cd App

Run:
streamlit run App.py

Open browser:
http://localhost:8501


Admin + User Login Notes
------------------------
1) Admin login:
- First time: Admin section open karo → “Create First Admin” form aayega
- New admin create kar ke login karo

2) User login:
- Analysis section me “User Login/Signup” aayega
- Signup karo → login karo
- Login ke baad analysis run kar sakte ho
- “My Reports” section me apne previous reports dikhेंगे


If App Doesn't Run (Common Fixes)
---------------------------------
1) NLTK error (stopwords/punkt missing):
   python -c "import nltk; nltk.download('stopwords'); nltk.download('punkt')"

2) MySQL connection error:
   - MySQL running check karo
   - DB credentials env vars check karo
   - Database `cv` exists check karo

3) reportlab/scikit-learn missing:
   pip install reportlab scikit-learn joblib scipy

4) Port already used:
   streamlit run App.py --server.port 8502


Optional: Exact environment versions
------------------------------------
If you want exactly same environment as developer PC:
- Use pip freeze output (you can generate again on your system):
  pip freeze > requirements-lock.txt
